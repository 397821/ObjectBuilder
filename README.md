### What is ObjectBuilder?

ObjectBuilder is a dat / spr files editor.

### Supported versions:

9.60 - 10.34

[ObjectBuilder 0.2] (https://www.dropbox.com/s/yo73u7ihziiyvxs/ObjectBuilder0.2.air)
