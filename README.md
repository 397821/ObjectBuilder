### What is ObjectBuilder?

ObjectBuilder is a dat / spr files editor.

### Supported versions:

10.32, 10.31, 10.30

### Download

[ObjectBuilder-0.1-Alpha1] (https://www.dropbox.com/s/jhs5k1gtugnphe0/ObjectBuilder-0.1-Alpha1.air)
