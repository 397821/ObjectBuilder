Object Builder

Object Builder is an editor for creating graphic objects for OTClient. You can add or edit new items, outfits, effects or missiles to your client.

Download .air View on GitHub

Features

  • It's faster to load or compile your projects.
  • Import or export your objects as sprite sheet.
  • Support for transparent sprites.
  • You're able to cut the imported images.
